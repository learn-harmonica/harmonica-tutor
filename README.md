# Harmonica Tutor

Beginner-friendly, interactive diatonic harmonica tutorial focused on Blues, Jazz, and Folk.

## Features (planned / in progress)

- Lessons for your diatonic harmonicas in keys C, G, and A
- Visual hole layout + tabs
- Curated external learning resources with citations
- Microphone-based note practice with honest feedback (local-only processing)

## Local development

- Install dependencies: `npm install`
- Run the dev server: `npm run dev`
- Run tests: `npm test`

## Privacy

The app is designed to process microphone audio locally in your browser. No audio uploads are required for core functionality.
